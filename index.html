Sejarah Moch Firza Alif Ramadhan

ditulis oleh Moch Firza Alif Ramadhan pada 2 mei 2024

Perkenalkan Nama saya Moch Firza Alif Ramadhan, Saya berasal dari Kota Malang, Saya lahir di kota Malang pada tanggal 07-10-2005, saya berasal dari daerah Jodipan yang berada di Kecamatan Blimbing tepat di tengah kota.

Di umur 5 tahun saya masuk di sekolah taman kanak-kanak sedari kecil hobi saya memang bermain saya sering memainkan permainan kayu malele bersama teman-teman saya setelah itu saya masuk di jenjang sekolah dasar(SD), disini saya mempunyai lebih banyak teman, di jenjang sekolah dasar saya sering mengikuti lomba festival albanjari untuk mewakili sekolah saya di tingkat kota, dan ketika saya lulus dari kelas 6 sekolah dasar orang tua saya memutuskan untuk memasukkan saya di pesantren karena pergaulan di rumah saya yang sangat bebas.

Dan saya melanjutkan SMP saya di pesantren, saya ditempatkan di SMP Al-Azhar, ketika pertama kali saya masuk pesantren saya langsung mendapatkan teman yang banyak dari seluruh indonesia, saya di pesantren selama 4 tahun dan saya selalu disuruh mengaji,sekolah,belajar agama,dan sholat berjamaah saya selalu senang melewati hal-hal dipesantren meskipun banyak peraturan yang harus di taati, dan setelah 4 tahun di pesantren saya merasa bosan di pesantren saya memutuskan untuk keluar dari pesantren dan sekolah di luar pesantren.

Setelah keluar dari pesantren saya memutuskan untuk masuk ke smk widyagama malang yang berada di jln borobudur, awal saya masuk sekolah saya mengikuti acara mpls di smk widyagama yang sangat meriah dan banyak siswa-siswa baru di smk widyagama.

setelah saya masuk teman saya menjadi lebih banyak lagi dari sebelumnya dan di awal saya masuk SMK widyagama saya mengikuti pelatihan Game bersama Letsplay dan Mengikuti global game jam untuk pertama kali, dan saya naik ke kelas 11 SMK, saya mengikuti pelatihan Digiup yang di Buat oleh Perusahaan Telkom untuk seluruh indonesia, setelah itu saya mengikuti Lomba Unipro di UNIKAMA(Universitas Kanjuruhan Malang).
